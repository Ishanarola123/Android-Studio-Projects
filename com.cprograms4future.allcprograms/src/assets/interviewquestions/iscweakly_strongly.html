Question:

Is C weakly typed language or strongly typed language and why?

Answer:
In C char data can be used in place of integers in case of arithmetic operations.
These kinds of features should not be permitted in strongly typed languages.
Hence, C is called as weakly typed by some.
For example, below program is valid in C

{

int a = 10;
char b = 'c';
printf("\n%d", a + b);
printf("\n%d", b);

}
