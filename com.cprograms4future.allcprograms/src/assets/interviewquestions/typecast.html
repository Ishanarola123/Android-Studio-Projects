Question:

What is typecast?

Answer:
Typecasting is for converting the datatype of a variable to another datatype
Example: float x=20.567,z=4.56;
Now two conditions
int y=(int)(x/z);------------------->where x/z=20.567/4.56=4.48
so due to typecasting float to int it will remove decimal part so,
y=4
Now what if we give
int y=(int)x/(int)z=----------------->(int)x=20(decimal part of x i.e 20.567 is removed) and (int)z=4
So y=20/4=5.
