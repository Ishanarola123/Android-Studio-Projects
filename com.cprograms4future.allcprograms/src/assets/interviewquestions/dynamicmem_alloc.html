Question:

What is dynamic memory allocation?

Answer:
 It means allocating memory at run time.Sometimes we don't know how much memory needs to be allocated for a variable
 Ex: When we don't know how many characters will be there in the Name of a person which is taken as input during runtime.It may be 10 characters 100 character or 200 characters

There are 4 dynamic memory allocation functions:
1.malloc()
2.calloc()
3.realloc()
4.free()
