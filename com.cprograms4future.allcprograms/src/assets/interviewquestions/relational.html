Question:

What are Relational Operators?

Answer:
Operator Description
== Used for comparing two operands.Returns true if both operands are equal
If A=6 and B=7
Then A==B returns false
!= Checks two operands and if both are not equal then it returns true
If A=6 and B=7
Then A!=B returns true
> Returns true if leftside operand is greater than right side operand
If A=6 and B=7
Then A>B returns false
< Returns true if leftside operand is lesser than right side operand
If A=6 and B=7
Then A<B returns true
>= Returns true if leftside operand is greater than or equal to right side operand
If A=7 and B=7
Then A>=B returns true
<= Returns true if leftside operand is lesser than or equal to right side operand
If A=7 and B=7
Then A<=B returns true
