Question:

Difference between getc,getchar,getch,getche,gets,puts?

Answer:
Function Meaning & Syntax Program
getc Reads character from any input stream
Syntax:
getc(File *Stream); 
#include<stdio.h>
main()
{
printf("%c", getc(stdin));
}
getchar Reads character from standard input stream
Syntax:
getchar(); 
#include<stdio.h>
main()
{
char c;
c=getchar();
printf("%c",c);
}
getch Non Standard function defined in conio.h and doesn't wait for enter key.
Entered character is not displayed unless displayed with printf as in given example
Syntax:
getch(); 
#include<stdio.h>
#include<conio.h>
main()
{
char c;
c=getch();
printf("%c",c);
}
getche Non Standard function defined in conio.h and doesn't wait for enter key.
Difference between getch and getche is Entered value is displayed on console. Where 'e' in getche stands for echoing(or displaying) the character entered
Syntax:
getche(); 
#include<stdio.h>
#include<conio.h>
main()
{
char c;
c=getche();
printf("%c",c);
}
gets Reads the string from user
Syntax:
gets(character_variable); 
#include<stdio.h>
main()
{
char c[20];
printf("Enter a String\n");
gets(c);
printf("%s",c);
}
puts Prints the string
Syntax:
puts(character_variable); 
#include<stdio.h>
main()
{
char c[20];
printf("Enter a String\n");
puts(c);
}
