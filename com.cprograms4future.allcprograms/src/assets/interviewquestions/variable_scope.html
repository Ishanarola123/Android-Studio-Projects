Question:

What is the scope of a variable in c?

Answer:
Within in a function block (simply to tell between tw braces in function)

Ex:
#include<stdio.h>
#include<string.h>
main()//called function
{
  int x=6;
  {
  	int x=25;
  	printf("x in 2nd part=%d\n",x);
  }
  printf("x in first part=%d\n",x);
}

Output:
x in 2nd part=25
x in first part=6
