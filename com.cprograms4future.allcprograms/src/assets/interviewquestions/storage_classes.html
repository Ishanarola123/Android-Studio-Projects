Question:

What are storage classes in C?

Answer:
1.Automatic Storage Class--Whenever a variable is declared this storage class is added by default whose scope is only inside a function or block
They have a garbage value by default

2.Static Storage Class---Their default value is zero and they have ability to store value even though they are out of scope as they are not redeclared when ever they appear
Check the example of Voting system under storage classes in this App

3.External Storage Class--When we declare a variable as extern then it is available globally through out the program.
Simply to tell it is also a global variable but can be declared inside a function also

4.Register Storage Class--These are used when we are using a variable more number of times to reduce execution time.
Usually these varibles are stored in the register of a processor which makes it to be accessible faster then normal variables.
