Question:

What is post and pre increment or decrement?

Answer:
int i;

i++ is post increment which will increment i by one after occurance of this operator i.e i=i+1
i-- is post decrement which will decrement i by one after occurance of this operator i.e i=i-1
++i is pre increment which will increment i by one immediately
--i is pre decrement which will decrement i by one immediately

Ex:-
int i=6;
printf("%d\n",i);
printf("%d\n",i++);
printf("%d\n",i);
printf("%d\n",++i);
printf("%d\n",i--);
printf("%d\n",i);
printf("%d\n",--i);

Output:
6
6
7
8
8
7
6

What is happening here
when you do
i++ then it wont return the incremented value but it will increment the i next time when it appears
like
i=6
c=i++; then c=6 but i=i+1=6
Now when c= i; again the value which is incremented at top will show here so c=7;
