Question:

What is implicit and explicit conversion?

Answer:
Implicit:
Suppose we are adding two numbers. The first number is of type int and the second number is of type float.
Since we cannot add an int and a float, so both the numbers have to be of the same data type i.e., either both are int or both are float.
Since float is a larger data type than int, therefore int variable gets converted into float automatically and then both the float variables add up.
This automatic conversion is called implicit conversion.

Explicit:
This is nothing but typecasting which is manual conversion of data type.
As int x=1;
float y=1.1;
then float j=(float)x+y;
here we are converting to float
