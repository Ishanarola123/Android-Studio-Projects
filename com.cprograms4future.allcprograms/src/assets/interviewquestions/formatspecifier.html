Question:

What is a format specifier?

Answer:
Format specifier indicates the compiler which type of value to be read from user at run time or to print when it ran the code. 
%c        -character
%s        -string(array of characters).
%d or %i  -integer
%f        -float
%lf       -double
%Lf       -long double
%ld       -long int
%o        – octal value.
%u        -unsigned int
%lu       -unsigned long int.
%x        – hex value.
