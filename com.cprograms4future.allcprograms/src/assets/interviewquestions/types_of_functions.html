Question:

What are the types of functions?

Answer:
There are 4 types:
1.Functions with no arguments(parameters) and no return type
2.Functions with no arguments(parameters) and has return type
3.Functions with arguments(parameters) and has no return type
4.Functions with arguments(parameters) and return type
