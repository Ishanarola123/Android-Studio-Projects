Question:

Advantages of gets() over scanf?

Answer:
scanf() ends taking input upon encountering a whitespace, newline or EOF gets() considers a whitespace as a part of the input string and ends the input upon encountering newline or EOF.
Ex: If we enter "Hello World" as input at run time and pressed Enter key Behaviour of scanf:-- Output will be only "Hello" Behaviour of gets:---Output will be "Hello World"
