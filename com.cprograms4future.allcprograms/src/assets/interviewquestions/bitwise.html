Question:

What are bitwise operators?

Answer:
Operator Description
"&" AND Operator used between two operands
Ex: X= A&B
X = 1 if Both A and B are true
  = 0 if any of the two A or B is false
"|" OR Operator used between two operands
Ex: X= A|B
X = 1 if any of the two A or B is true
  = 0 if Both A and B are false
"^" XOR Operator used between two operands
Ex: X= A^B
X = 1 if A and B are different(i.e if( A=1, B=0 )or (A=0, B=1)
    = 0 if Both A and B are Same (i.e if(A=0,B=0) or (A=1,B=1)
"~" It is Binary 1's Complement
If A=1100
~A=0011
">>" It is Right Shift Operator
If A=8 (i.e 1000 in binary
) Then A>>2 will shift the binary right side by 2 times
So A=2(i.e 0010) right side 2 zeros are removed
"<<" It is Left Shift Operator
If A=2 (i.e 10 in binary
) Then A<<2 will shift the binary left side by 2 times appending zeros at right side
So A=8(i.e 1000) right side 2 zeros are appended which means it is shifted to left by 2 times
