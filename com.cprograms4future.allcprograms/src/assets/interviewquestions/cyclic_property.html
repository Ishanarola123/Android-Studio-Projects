Question:

What is cyclic property of datatype in C?

Answer:
            #include<stdio.h>
            main()
            {
            char c=129;
            char d=-129;
            printf("c=%d,d=%d\n",c,d);

            }
Ouput:-
c=-127,d=127

It is because of overflow of storage as "char" variable can store only from -128 to +127.And if it crosses it uses cyclic property. So 129 crosses 127 and it goes to the next numbers clockwise as shown in diagram which is -127 i.e 129-127=2 so it chooses the 2 value after 127
Similarly -129 crosses -128 by 1 so it chooses the next value after -128 anti-clockwise which is +127.
This how we got output.
