Question:

What is Dangling Pointer?

Answer:
Dangling pointer is the pointer which has an invalid memory location.
Usually this happens in such situations where we free memory allocated to pointer from memory location then it is pointing to an invalid memory location
Ex:
int *i=NULL;
{
  int j=2;
  i=&j;
}
j has scope within braces but out of that it is not at all existed then at what pointer 'i' is locating at.It is locating to invalid memory location.
