Question:

What are the modes in files?

Answer:
Mode Description
"r" Opens file in reading mode.If file is found then the file is loaded into memory and pointer is set to the first character of file. If file doesn't exist or cannot be opened it returns NULL
"w" If file is exsted then its content is overwritten.
If file doesn't exist then it will create a new file and writes the data into it. If file can't be opened in any case it will return NULL
"a" Used to add new content at the end of file.If file is existed then the new data/content is appended at the end ot file.
In this case the file is loaded into memory and pointer is set to last character of the file.
If file is not existed then a new file is created.
If in case file can't be opened then it returns NULL
"r+" Opens existing file in reading and writing mode.If file can't be opened then it returns NULL
"w+" Opens a new file in reading and writing mode if file is not existed.
If file is existed the content is overwritten.
If file can't be opened then it returns NULL
"a+" Opens file in reading and appending mode if file exists.
If file can't be opened returns NULL.Cannot modify existing data,But can apend the data at the end of file.
