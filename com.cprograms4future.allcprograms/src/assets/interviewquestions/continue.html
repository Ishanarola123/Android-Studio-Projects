Question:

What is "continue" Statement?

Answer:
This keyword is used to skip the statements below continue statement and go to next iteration.
To skip 5 and 7 while printing from 1 to 10

#include<stdio.h>
main()
{
int i;
    for(i=1;i<=10;i++)
       {

         if(i==5||i==7)
         continue;
         printf(“%d\n”,i);
      }
}
