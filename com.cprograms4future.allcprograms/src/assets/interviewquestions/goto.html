Question:

What is "goto" statement?

Answer:
This keyword is used when ever you want to jump from one statement to the other where there is a label.For every goto there must be a label from where it starts
Syntax:
label:
……………….;
statements;
…………………;
goto label;

example:
#include<stdio.h>
main()
{
  int i,j;
  for(i=1;i<10;i++)
  {
	if(i==5)
	goto next;
	else
	printf(“%d\n”,i);
  }
  next:
  for(j=11;j<=20;j++)
  {
	printf(“%d\n”,j);
  }
}
