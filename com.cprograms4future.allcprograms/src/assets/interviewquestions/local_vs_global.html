Question:

What is the difference between local and global variables?

Answer:
Global variables are those which are available through out the program
But Local variables are only available between the block of the functions i.e between braces
Ex:
 #include<stdio.h>
int x=5;
void hello();
main()
{
    int y=35;
    printf("Global in Main x=%d\nLocal in Main y=%d\n",x,y);
    hello();
}

void hello(){
	int y=55;
	printf("Global in Hello function x=%d\nLocal in Hello function y=%d\n",x,y);
}
Output:
Global in Main x=5
Local in Main y=35
Global in Hello function x=5
Local in Hello function y=55
