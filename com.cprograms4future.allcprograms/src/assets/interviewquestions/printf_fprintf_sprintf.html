Question:

Difference between printf,fprintf and sprintf

Answer:
1.printf:-Used to print the characters on console
Ex: printf("Hello World");

2.sprintf:-Used to store the characters on a character buffer specified inside function
Ex: 
#include<stdio.h>
main(){
char hello[100];
int totalPrograms=350;
sprintf(hello,"Hey this is C Programs app with %d + programs\n",totalPrograms);
printf("%s",hello);
}
Output: Hey this is C Programs app with 350 + programs

3.fprintf: Used to print the character in file instead on console
Ex:
#include<stdio.h>
main(){
  char language[100];
  FILE *file;
  file=fopen("fprintfExplanation.txt","w");
  if(file == NULL)
  {
  	printf("File Not Found\n");
  }
  else
  {
  	printf("Enter a language you like\n");
  	gets(language);
  	fprintf(file,"I like %s",language);
  }
  fclose(file);

}
Output:
Enter a language you like
C Language
Now open text file you could see
I like C Language
