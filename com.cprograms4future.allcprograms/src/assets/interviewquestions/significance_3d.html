Question:

Significance of %2d or %3d and %-3d etc

Answer:
1.If we want to take only 3 digits out of given 6 digits as input at run time

Ex: int x;
scanf("x value is %3d",&x);

Output:
If we enter 345678
then output is
x value is 345

2. %6d means the output will be filed in 6 columns and if the output contains 6 characters or digits then it won't have any affect
but if the output is having less than 6 characters or digits then the value is right justified and is padded with blanks on the left.
Ex:
#include<stdio.h>
main()
{
int i=30,j=35;
printf("%d,%6d\n",i,j);
}
output:
30, 35

So if we use in scanf it will take those number of digits as input
if we use it in printf it will add spaces beside the first digit
3.)%-6d is same as %6d but the difference is left justification is desired and the value will be padded with blanks on the right
