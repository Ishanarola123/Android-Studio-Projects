Question:

What are the rules for a variable?

Answer:
  1. It can have numbers,digits.
  2. Should not contain spaces between 2 words but we can append them using underscore.
    Ex:-Hello_world…….correct way.
    Hello world……..wrong way.
  3. Never start with a digit.
  4. They are case sensitive which means A and a are not same.
    Ex:- int HeLLo_World=256;
    int Hello_World=341;
    So if we print HeLLo_World then the output is 256 and if we print Hello_World then the output is 341.
