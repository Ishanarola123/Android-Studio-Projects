Question:

What is enumeration?

Answer:
It is a keyword used to assign name to integral constants.
Ex:
1.If we want to assign by default from zero
enum week{monday,tuesday,wednesday,thursday,friday,saturday,sunday};
So monday=0,tuesday=1 ans so on...

2.If we want to assign from certain number till the end continuing from the given starting number
enum week{monday=4,tuesday,wednesday,thursday,friday,saturday,sunday};
Then monday=4,tuesday=5,wednesday=6 and so on....

3.)If we want to assign separately
enum week{monday=24,tuesday=23,wednesday=31,thursday=42,friday=51,saturday=66,sunday=79};
